# BuyApp
Brokerage system between sellers and buyers Based PHP & SQL.
<hr>
The system created with HTML, CSS, BOOTSTRAP 4, JavaScript, PHP, SQL ,MYSQL DataBase.
<hr>
<b>Buyer</b>:
<ul>
  <li>buy products</li>
  <li>Select categories to fit products that interest the buyer</li>
  <li>Track the buyer's bids and accepted products</li>
</ul>
<hr>
<b>Seller</b>:
<ul>
  <li>Sell products</li>
  <li>Accept bids from buyers</li>
  <li>Manage the products</li>
  <li>Track products's sells</li>
</ul>
<hr>
<b>Admin</b>:
<ul>
 <li>Admin panel to track statistics and manage the app</li>
</ul>
